# 🌦️ Weather App

A simple and responsive weather application built while learning **JavaScript**, **API integration**, and **DOM manipulation**.  
It fetches real-time weather data and displays it with dynamic UI updates.

## 🚀 Features

- 🔍 Search weather by city name  
- 🌡️ Real-time temperature display  
- 💧 Humidity information  
- 🌬️ Wind speed details  
- 🌤️ Dynamic weather icons  
- ⚠️ Error handling for invalid city names  
- 🎨 Clean and responsive UI  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- OpenWeather API  

---

## ⚙️ How It Works

1. User enters a city name  
2. App sends request to OpenWeather API  
3. Receives JSON weather data  
4. Updates UI dynamically
